# Campus Food Ordering (Frontend Only)

## Overview

This is a beginner-level frontend project built using HTML, CSS, and JavaScript. It simulates a simple campus food ordering interface with a menu, cart system, and a basic admin view.

## Tech Stack

* HTML
* CSS
* JavaScript (Vanilla)
* localStorage (for temporary state management)

## Features

* Food menu display
* Add/remove items from cart
* Basic cart total calculation
* Simple admin panel (frontend-only logic)
* Data stored using browser localStorage

## Important Note

This project does NOT include:

* Backend server
* Database integration
* Real authentication or security

The admin login is only for frontend demonstration and is not secure.

## Learning Purpose

Built to practice DOM manipulation, UI design, and state handling using JavaScript.

## Future Improvements

* Add backend using Node.js or Python (FastAPI/Flask)
* Connect real database (Supabase/PostgreSQL)
* Implement proper authentication
